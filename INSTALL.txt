Building deskvox and vconv
==========================

To build the static virvo library and the deskvox application do the following:
./autogen.sh
make

You can pass additional options to the configure script to autogen.sh.


Building the virvo library
==========================

Go to the virvo subdirectory and use these commands:
./autogen.sh
make

You can pass additional options to the configure script to autogen.sh.
